# Cache Pilot

Project for studies, focusing on implementing functions for cache management.

## Technologies

* .NET 8

* Blazor

* Redis

## Functionalities

* Manage users

## Run

Clone the project:
```bash
git clone https://github.com/diogoandrad/cache-pilot.git
```

Enter the project folder:
```bash
cd cache-pilot
```

Run the application:
```bash
dotnet run
```
